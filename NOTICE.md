## On AGPL §13 (network use)

If you run a modified version of this software on a server, the AGPL
requires you to provide the **full source code of your modified
version** to all users interacting with it over a network — including
the modifications you made.

This is intentional. It exists so that improvements to OmniDiff flow
back to the community rather than being captured privately by SaaS
operators.
